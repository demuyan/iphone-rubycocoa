RubyCocoa - A Ruby/Objective-C Bridge for Mac OS X with Cocoa

What's RubyCocoa ?

RubyCocoa is a Mac OS X framework that allows Cocoa programming in the object-oriented scripting language Ruby. Please visit the RubyCocoa Web Site (http://rubycocoa.sourceforge.net/) for getting more detail.

Items Installed With This Package

After installation is completed, the following items are installed. Please try to launch the RubyCocoa application '/Developer/Examples/RubyCocoa/SimpleApp.app' which is contained in the Examples Folder.

  • RubyCocoa framework (Core) - '/Library/Frameworks/RubyCocoa.framework'
  • RubyCocoa library (stub) - in the '/usr/lib/ruby/site_ruby/1.6/osx/'
  • RubyCocoa extended library (stub) - '/usr/lib/ruby/site_ruby/1.6/powerpc-darwin7.0/rubycocoa.bundle'
  • Some templates for Xcode - in the '/Library/Application Support/Apple/Developer Tools/'
  • Some templates for ProjectBuilder - in the '/Developer/ProjectBuilder Extras/'
  • Documents folder (html) - '/Developer/Documentation/RubyCocoa/'
  • Examples folder - '/Developer/Examples/RubyCocoa/'
