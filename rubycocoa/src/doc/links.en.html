Let's make an application with RubyCocoa (Japanese)
Tutorial for RubyCocoa. It describes programming in RubyCocoa through making a simple calculator application.
An Introduction to RubyCocoa, Part 1 (English) , Part 2

Dr. Dobb's Journal May 2002
an article about RubyCocoa written by Chris Thomas.
RubyAEOSA
Ruby interface for Apple Event and Open Scripting Architecture (OSA) on Mac OS X based Carbon framework. Recently, some classes (e.g. NSAppleScript) have been added to Cocoa incompletely.
CocoaRBBR.app (Japanese)
The application which implemented Ruby class browser with RubyCocoa.
uptime
This is a GUI application for using .Mac as blog implemented with RubyCocoa.

Same Kind of RubyCocoa

F-script
"F-Script is a lightweight object-oriented scripting layer specifically designed for Mac OS X object system". Links page have many links to same kind of RubyCocoa.
CamelBones
"An Objective-C/Perl bridge framework". Mac OS X contains PerlObjCBridge that support only Foundation framework. Camelbones allows writing a Cocoa application in Perl.
PyObjC
"The Python <-> Objective-C Bridge". PyObjC project want to offer the same level of support for GNUstep. It contrasts with the fact that RubyCocoa is closed in Mac OS X currently.
AppleScript Studio
AppleScript Studio combines an application framework with a development environment.
Ruby GNUstep
RIGS is a Ruby interface to the GNUstep development environment and indirectly to the Objective C language.
Ruby/.NET Bridge
The Ruby/.NET Bridge lets you use .NET and Ruby objects together in your programs. You can access .NET objects from Ruby and vice-versa.
