Ruby 1.6.8 で config エラーを直すパッチ (2003-01-12)

Ruby 1.6.8 で RubyCocoa 0.4.0 をソースから構築するときに、 'ruby install.rb config' を実行すると Segmentation fault が発生します。 この問題を解決するための パッチ を置きました。パッチの当て方は以下のとおりです。

$ cd {rubycocoa-0.4.0}
$ patch -p0 < ruco0.4.0-fw-post-config.patch
