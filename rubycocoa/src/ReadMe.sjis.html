RubyCocoa - A Ruby/Objective-C Bridge for Mac OS X with Cocoa

RubyCocoaとは？

RubyCocoaは、オブジェクト指向スクリプト言語RubyでのCocoaプログラミングを可能とする、Mac OS Xのフレームワークです。詳しくはRubyCocoaホームページ(http://rubycocoa.sourceforge.net/)をご覧ください。

このパッケージでインストールされるもの

インストールが完了すると以下のものがインストールされます。サンプルフォルダに入っているRubyCocoaアプリケーション'/Developer/Examples/RubyCocoa/SimpleApp.app'が起動できるか確認してみてください。

  • RubyCocoaフレームワーク本体 - '/Library/Frameworks/RubyCocoa.framework'
  • RubyCocoaライブラリ(stub) - '/usr/lib/ruby/site_ruby/1.6/osx/'の中
  • RubyCocoa拡張ライブラリ(stub) - '/usr/lib/ruby/site_ruby/1.6/powerpc-darwin7.0/rubycocoa.bundle'
  • Xcodeのテンプレート - '/Library/Application Support/Apple/Developer Tools/'の中
  • ProjectBuilderのテンプレート - '/Developer/ProjectBuilder Extras/'の中
  • ドキュメント(HTML) - '/Developer/Documentation/RubyCocoa/'
  • RubyCocoaサンプルフォルダ - '/Developer/Examples/RubyCocoa/'
